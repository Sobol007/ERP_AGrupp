Мобильное приложение "iCRM" позволяет работать с корпоративным решением "1C:CRM" с мобильного устройства.

Инструкция по установке:

Скачайте приложение "iCRM" на свой телефон из магазина.

Ссылка на Google Play: https://goo.gl/Dg27oo

Ссылка на App Store: https://goo.gl/Sk1sgW



Если это первый запуск приложения:

  1. Откройте приложение.

  2. Выберите пункт: Настроить синхронизацию.

  3. В открывшемся окне выберите пункт «Присоединиться к синхронизации».

  4. В открывшемся окне выберите пункт «Корпоративная синхронизация».

  5. Укажите E-mail, на который вы получили код приглашения, нажмите кнопку «Продолжить».

  6. Вам будет выслано дополнительное письмо для подтверждения адреса электронной почты. Укажите код в специальном поле, нажмите кнопку «Подтвердить».

  7. В открывшемся окне укажите код приглашения, для этого скопируйте его из письма.

  8. Если код приглашения указан верно, начнется загрузка данных. После завершения загрузки данных, с приложением можно начинать работать.

  1. Перейдите на закладку "Синхронизация" - "Настройки синхронизации"

  2. В открывшемся окне выберите пункт «Присоединиться к синхронизации».

  3. В открывшемся окне выберите пункт «Корпоративная синхронизация».

  4. Укажите E-mail, на который вы получили код приглашения, нажмите кнопку «Продолжить».

  5. Вам будет выслано дополнительное письмо для подтверждения адреса электронной почты. Укажите код в специальном поле, нажмите кнопку «Подтвердить».

  6. В открывшемся окне укажите код приглашения, для этого скопируйте его из письма.

  7. Если код приглашения указан верно, начнется загрузка данных. После завершения загрузки данных, с приложением можно начинать работать.
