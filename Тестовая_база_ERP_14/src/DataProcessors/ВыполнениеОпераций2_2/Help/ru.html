Выполнение операций (2.2)

Рабочее место предназначено для осуществления действий по выполнению производственных операций.

В общем случае на форме представлены 2 закладки: Выполнение операций и Операции. Если в подразделении используются сменные задания, то назначение операций выполняется в рабочем месте Формирование сменных заданий, а в форме отображаются только данные закладки Операции.

Выполнение операций

На закладке отображается состояние объема работ, подлежащего выполнению при помощи операций, образованного этапами в статусе Начат. Список содержит операции этапов и колонки, отражающие количественное состояние выполнения.

  • Запланировано - все запланированное количество операции. Содержимое колонки не меняется по мере выполнения.
  • Ожидает создания документа "Производственная операция" - количество операции, которое еще не принято в работу и по которому не создан документ Производственная операция.
  • Ожидает предшествующие - количество операции, которое пока еще нельзя принять в работу, так как не начаты предыдущие операции.
  • Начаты предшествующие - на указанное количество начаты предшествующие операции.
  • Можно выполнять - на указанное количество предшествующие операции завершены либо нет предшествующих.
  • Выполняется - количество взятых в работу, но еще не завершенных операций.
  • Выполнено - количество завершенных операций.

Для управления выполнением служат команды:

  • Создать - по выбранной строке создается документ Производственная операция в статусе Создана. Может применяться для заблаговременного создания документов операций.
  • Принять в работу - по выбранной строке создается документ Производственная операция в статусе Выполняется с назначением исполнителя.
  • Отметить выполнение - по выбранной строке отмечает завершение выполнения ранее принятых в работу операций. Если операция не принималась в работу, то создается документ Производственная операция в статусе Выполнена с назначением исполнителя.

Операции

На закладке отображаются список документов Производственная операция. С помощью этой закладки можно детально контролировать ход выполнения операций и отмечать их выполнение.
