Документы приема в переработку (к оформлению)

Рабочее место предназначено для оформления документов приема в переработку по распоряжениям (Заказам давальцев).

Каждой вкладке соответствует вид оформляемого документа: Поступление сырья от давальца, Передача давальцу, Отчет давальцу, Возврат сырья давальцу.
На вкладке отображается список распоряжений, по которым необходимо оформить документы приема в переработку.

Состояния оформления


Для индикации состояния оформления документов и расходных/приходных ордеров на товары, предусмотрены соответствующие информационные колонки с пиктограммами.
Пояснение назначения пиктограмм приведено в соответствующих колонкам отборах.

Оформление документов


Документы могут быть оформлены по одному или нескольким выделенным заказам.

По команде Оформить по заказам создается документ и заполняется всеми позициями заказов, по которым еще не оформлены документы.
По команде Оформить по отгрузе (приемке) создается накладная и заполняется всеми отгруженными (принятыми) по ордерам позициями, при этом, если накладные уже были созданы ранее, система предложит их скорректировать в отдельной форме.
