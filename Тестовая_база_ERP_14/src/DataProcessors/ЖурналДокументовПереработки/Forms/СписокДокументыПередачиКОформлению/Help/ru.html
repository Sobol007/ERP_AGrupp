Документы передачи в переработку (к оформлению)

Рабочее место предназначено для оформления документов передачи в переработку по распоряжениям (Заказам переработчикам).

Каждой вкладке соответствует вид оформляемого документа: Передача сырья переработчику, Поступление от переработчика Отчет переработчика, Возврат сырья от перереработчика.
На вкладке отображается список распоряжений, по которым необходимо оформить документы передачи в переработку.

При использовании Партионного учета версии 2.2, при оформлении поступления без заказа, отчет оформляется по переработчику в целом и в рамках одной организации. В такой ситуации на странице Отчет переработчика строки распоряжения появляются в списке при наличии не отраженного отчетом переработчика остатка продукции, полученной документами поступления.

Состояния оформления


Для индикации состояния оформления документов и расходных/приходных ордеров на товары, предусмотрены соответствующие информационные колонки с пиктограммами.
Пояснение назначения пиктограмм приведено в соответствующих колонкам отборах.

Оформление документов


Документы могут быть оформлены по одному или нескольким выделенным заказам.

По команде Оформить по заказам создается документ и заполняется всеми позициями заказов, по которым еще не оформлены документы.
По команде Оформить по отгрузе (приемке) создается накладная и заполняется всеми отгруженными (принятыми) по ордерам позициями, при этом, если накладные уже были созданы ранее, система предложит их скорректировать в отдельной форме.
