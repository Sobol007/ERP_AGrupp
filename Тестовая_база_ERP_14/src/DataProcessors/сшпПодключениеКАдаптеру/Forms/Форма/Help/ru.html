Обработка "Подключение к адаптеру" предназначена для создания настроек подключения системы к адаптеру ESB (т.н. создание точки подключения).

Для создания подключения необходимо указать:

  • Формат обмена - формат, в который будут сериализовываться исходящие сообщения для отправки в ESB (тип - перечисление "Форматы сообщений": XML, JSON, FastInfoset);
  • Сервер адаптеров - имя сервера адаптеров (тип - Строка);
  • Имя адаптера - имя адаптера, к которому происходит подключение (тип - Строка);
  • Тип коннектора - протокол, по которому будет происходит подключние и отправка сообщений в адаптер  (тип - перечисление "Типы коннекторов ESB": REST, SOAP).

 

Для сохранения настроек не нажмите кнопку "Сохранить параметры подключения".
