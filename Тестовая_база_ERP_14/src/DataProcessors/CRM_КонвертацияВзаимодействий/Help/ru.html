Обработка "Конвертация взаимодействий"

Назначение

Конвертация документов Встреча, Телефонный звонок и Запланированное взаимодействие во Взаимодействия CRM.

Внимание! Конвертацию отменить нельзя, настоятельно рекомендуется предварительно сделать архивную копию базы  и сначала проверить конвертацию на копии!

При повторном выполнении конвертации будут созданы новые документы.

 

Описание использования

  1. Укажите дату, начиная с которой будут обрабатываться документы. Если дата обработки не указана, будут обработаны все документы.
  2. Укажите Вид взаимодействия для каждого вида документа:
    • Вид взаимодействия для Встреч - вид взаимодействия для конвертации  документов "Встреча" (рекомендуется вид  "Встреча")
    • Вид взаимодействия для Звонков - вид взаимодействия для конвертации  документов "Телефонный звонок" (рекомендуется вид  "Телефонный звонок (исходящее)")
    • Вид взаимодействия для Прочих - вид взаимодействия  для конвертации  документов "Запланированное взаимодействие" (рекомендуется вид  "Прочее")
  3. Нажмите кнопку "Выполнить конвертацию".
    При большом объеме документов обработка может выполняться длительное время.
