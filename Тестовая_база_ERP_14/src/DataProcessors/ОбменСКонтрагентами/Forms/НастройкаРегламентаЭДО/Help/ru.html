Настройка регламента ЭДО

Используется для настройки регламента обработки электронного документа.

  • Профиль настроек ЭДО - профиль, по которому будет выполняться обмен по выбранному виду документа. Используется только при вызове из формы справочника Настройки ЭДО.
  • Подписывать документ (по маршруту) - задает необходимость подписания электронного документа. Если флаг недоступен для изменения, это означает, что документы выбранного вида подписываются в обязательном порядке. Если флаг установлен, есть возможность выбрать маршрут, задающий требования к подписанию документа.
  • Ожидать извещение о получении - если установлен, документооборот не будет завершен, пока получатель не отправит извещение о получении.
  • Ожидать ответную подпись - определяет, должен ли подписываться электронный документ получателем.

В зависимости от формата электронного документа доступно изменение настроек формирования документа.

  • Выгружать в качестве кода товара - задает вариант заполнения кода товара в электронном документе.
  • Настройки заполнения дополнительных полей - отображает наличие настроек заполнения дополнительных полей. При переходе по гиперссылке открывается мастер настройки дополнительных полей по выбранному формату электронного документа.
