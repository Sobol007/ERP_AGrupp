Задачи "Задачи по ВЭД"

Панель задач вызывается в интерфейсе "Внешнеэкономическая деятельность" из пункта меню «Сервис» — «Задачи по ВЭД».

Таймер обратного отсчета срока жизни задачи в календарных или рабочих днях задается на форме "Сроки представления документов валютного контроля", вызываемой через одноименный пункт меню «Настройка».

Список задач по внешнеэкономической деятельности содержит задачи, которые необходимо выполнить пользователям с ролями "Валютный контролер" и "Юрист". Поэтому для задействования механизма задач по ВЭД, оператору следует присвоить такие роли.

Виды задач, которые будут добавляться панель задач ВЭД, определяются фактом создания расчетных документов с нерезидентами, служащих основанием для формирования документов валютного контроля.

Задачи на формирование документа Заявление о постановке на учет контракта (ЗПУК) создаются при создании в программе контрактов ВЭД на основе договоров.

Задачи на формирование документа Справка о кодах видов операций (СКВО) создаются при вводе в программу расчетных банковских документов.

Задачи на формирование документа Справка о подтверждающих документах (СПД) создаются при вводе в программу расчетных документов с контрагентами: документов по оприходованию от поставщиков (Поступление товаров и услуг, ГТД по импорту и др.) и документов отгрузки покупателям (Реализация товаров и услуг и др.).

Документ валютного контроля (ЗПУК, СКВО, СПД) можно сформировать непосредственно из списка задач по расчетному документу, породившему задачу. Для этого надо щёлкнуть правой клавишей мыши по строке с задачей и в распахнувшемся контекстном меню выбрать пункт "Выполнить задачу". При этом надо иметь в виду, что в отличие от стандартного пакетного режима создания документов валютного контроля, режим работы из панели задач позволяет сформировать лишь один документ валютного контроля по одному расчетному документу.

Жизненный цикл задачи отмечается её статусом с иконкой соответствующего цвета:

  • при создании задачи по формированию документа валютного контроля иконка имеет белый цвет: ;
  • в момент проведения документа валютного контроля на иконке белого цвета появляется "галочка" зелёного цвета: :
  • после вывода документа валютного контроля на печать или выгрузки в систему клиент-банк иконка с "галочкой" меняет цвет на зелёный, что означает, что задача выполнена: .
  • если справка валютного контроля проведена, но после этого расчетный документ, по которому она создана, перепроведен, то красный контур вокруг иконки белого цвета с зелёной "галочкой" будет означать, что в расчетном документе изменились ключевые реквизиты (дата, номер, сумма и т.п.), которые, возможно, потребуют создания корректировочной справки валютного контроля: ;
  • если справка валютного контроля распечатана (выгружена в систему клиент-банк), но после этого расчетный документ, по которому она создана, перепроведен, то красный контур вокруг иконки зелёного цвета будет означать, что в расчетном документе изменились ключевые реквизиты (дата, номер, сумма и т.п.), которые, возможно, потребуют создания корректировочной справки валютного контроля: .

В случае невыполнения задачи в течение длительного времени и приближении срока представления (дедлайна) в банк УК документа валютного контроля, иконка задачи будет менять цвет на различные степени тревожности следующим образом:

  • за два дня до дедлайна - жёлтый цвет: ;
  • за один день до дедлайна - оранжевый цвет: ;
  • в момент дедлайна - красный цвет: ;
  • в случае просрочки - чёрный цвет: .

 

При изменении состава видов задач, существующие задачи остаются в списке. Ненужные задачи можно непосредственно удалить с помощью комбинации клавиш Shift+Del.

После выполнения задачи она исчезает из списка автоматически на следующий день после дедлайна.

При выборе задач доступен множественный выбор задач (с помощью кнопок Ctrl+Shift).
