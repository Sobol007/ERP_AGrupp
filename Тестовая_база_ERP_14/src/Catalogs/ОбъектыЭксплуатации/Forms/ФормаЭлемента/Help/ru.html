Объект эксплуатации

Справочник предназначен для регистрации объектов хозяйственной деятельности с необходимой детализацией административной и технологической структуры. Отдельные объекты эксплуатации могут быть приняты к учету как основные средства, при этом в программе соблюдается регламентировано установленный порядок их принятия к учету и других изменений в параметрах учета.

Объекты эксплуатации используются для организации ремонтной деятельности, используются в заказах на ремонт. В случае использования подсистемы ремонтов (при включенной опции НСИ и администрирование – Настройка НСИ и разделов – Производство – Ремонты – Управление ремонтами) в карточке объекта эксплуатации появляются дополнительные реквизиты и настройки: Статус, группы полей Ремонт и Эксплуатация.
Детализация технологической структуры объекта эксплуатации осуществляется с помощью описания структуры узлов по навигационной ссылке Узлы объекта.

Для узлов возможно указать собственные классы обслуживания и независимо фиксировать наработку. Для узлов сохраняется история изменения принадлежности к объекту эксплуатации – перенос из состава одного объекта эксплуатации в состав другого. При таком переносе сохраняется наработка, накопленная узлом в составе первоначального объекта эксплуатации.

Закладка Основные


На закладке определяются основные реквизиты и параметры учета объекта эксплуатации.

  • Поле Класс обязательно к заполнению, при использовании подсистемы управления ремонтами. При необходимости, основная классификация уточняется Подклассом;
  • Направление деятельности - сквозной аналитический разрез при планировании и учете доходов, расходов;
  • Паспортные характеристики не обязательны для заполнения, служат для целей детализированного описания и классификации объекта эксплуатации;
  • Реквизиты группы Ремонт Статья расходов и Способ отражения зарплаты - при выполнении ремонтов материальные затраты будут отнесены на статью расходов (тип аналитики расходов статьи должен быть "Заказ на ремонт"), а трудозатраты приняты к учету в соответствии со способом отражения;
  • В группе Эксплуатация указывается Ответственное подразделение и Расположение - строка в произвольном формате для упрощения поиска объекта;
  • Реквизиты группы Регламентированный учет не обязательны к заполнению, если на момент заполнения не известны параметры классификации объекта для регламентированного учета они могут быть указаны непосредственно при Принятии к учету ОС;
  • Недвижимое имущество - Признак того, что основное средство является видом имущества, признаваемым в законодательном порядке недвижимым. Используется при расчете налога на имущество. Признак недоступен для группы ОС "Рабочий скот" и "Продуктивный скот". Земельные участки всегда являются недвижимым имуществом. Для остальных групп признак можно изменить.
  • Для целей классификации в рамках Международного учета требуется указание Группы ОС.
  • Если установлен признак Транспортное средство зарегистрировано в реестре системы "Платон", то транспортное средство доступно для выбора в документах Отчеты оператора системы "Платон" и имеет особый порядок расчета транспортного налога. Признак доступен если включена опция НСИ и администрирование - Внеоборотные активы - Учет платежей в системе "Платон" и выбрана группа Транспортные средства, Машины и оборудование (кроме офисного).

Закладка Параметры учета наработок


Закладка предназначена для заполнения свойств показателей наработок данного объекта эксплуатации. В случае, если заданы соответствующие настройки для перечня контролируемых показателей на этапе классификации, для каждого показателя наработки объекта можно задать Источник получения текущих значений наработки и Назначенный ресурс.

Закладка Рабочие центры


На закладке уточняется список зависимых рабочих центров, информация об ограничении доступности которых, будет предоставлена для анализа при проведении ремонтных работ на объекте эксплуатации.


Закладка Драгоценные материалы


На закладке предусмотрена возможность перечисления драгоценных материалов, входящих в состав основного средства. Доступ к этому разделу регулируется функциональной опцией Драгоценные металлы и камни в разделе НСИ и администрирование - Номенклатура - Дополнительная информация.

Допускается выбирать единицы измерения, у которых тип измеряемой величины Вес.
Например, чтобы указать 0,000001 (г) можно выбрать единицу измерения миллиграмм и указать 0,001 (мг).
Если нужной единицы измерения нет, то ее можно создать и указать нужную кратность.
