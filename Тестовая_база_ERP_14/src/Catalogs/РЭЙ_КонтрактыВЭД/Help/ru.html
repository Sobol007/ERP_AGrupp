Справочник «Контракты ВЭД»

Справочник вызывается для ввода и редактирования информации в интерфейсе "Внешнеэкономическая деятельность" из пункта меню "Контракты ВЭД" в разделе "Справочники".

Справочник предназначен для хранения дополнительных реквизитов договоров, заключенных организацией-резидентом с контрагентами-нерезидентами Российской Федерации. Данный справочник является расширением справочника Договоры контрагентов для целей внешнеэкономической деятельности.

Для получения доступа к дополнительным реквизитам в карточке договора используется кнопка "ВЭД", нажатие которой в первый раз инициирует процесс регистрации договора в качестве контракта ВЭД. При этом в карточку контракта ВЭД будут оттранслированы основные данные из карточки договора. Остальные дополнительные реквизиты в карточке контракта ВЭД должны быть заполнены пользователем.

Наименование контракта ВЭД будет сформировано программой автоматически после ввода пользователем номера, даты подписания и предмета контракта. При необходимости пользователь может скорректировать предложенное программой наименование. После ручной корректировки наименования режим автоматического формирования наименования контракта ВЭД отключается. Для повторного включения такого режима следует очистить поле "Наименование".

Если контракт ВЭД не имеет номера, следует установить флаг "Без номера". Аналогично, если сумма контракта не определена, что характерно для рамочных контрактов, следует установить флаг "Без суммы".

При любом последующем нажатии кнопки "ВЭД" из карточки договора будет открываться карточка контракта ВЭД. После любых изменений данных в карточке контракта ВЭД необходимо пересохранить карточку договора, чтобы произведенные изменения вступили в силу.

 

В карточке контракта ВЭД по умолчанию имеются две закладки: "ВЭД" и "Банк. счета".

Закладка «ВЭД»

На закладке по умолчанию имеются вложенная закладка "Условия".

Вложенная закладка "Условия"

На закладке указываются основные условия, необходимые для заполнения документа Заявление о постановке на учет контракта.

Вложенная закладка "Разрешения"

Закладка по умолчанию недоступна. Для ее появления на вложенной закладке "Условия" закладки "ВЭД" следует установить флаг "Разрешения органов валютного контроля на осуществление валютных операций". После этого следует заполнить табличную часть информацией о разрешениях.

Закладка «Банк. счета»

Для обеспечения анализа банковских документов на закладке "Банк. счета" следует указать платежные реквизиты - все банковские счета организации, через которые будут производиться платежи по контракту ВЭД.

Для контракта ВЭД, по которому планируется получение или списание сумм по счетам в банках-нерезидентах, в перечень банковских счетов необходимо включить хотя бы один счет в банке УК, даже если он явным образом не указан в самом контракте.

 

Для контракта ВЭД, по которому планируется получение валютных сумм, в перечень банковских счетов необходимо включить транзитный валютный счет, даже если он явным образом не указан в самом контракте.

Закладка «Стороны»

Закладка по умолчанию недоступна. С целью вывода в пункт 2 "Реквизиты нерезидента (нерезидентов)" документа Заявления о постановке на учет контракта и отчета Ведомость банковского контроля нескольких контрагентов по многостороннему контракту следует установить флаг "Многосторонний" и в открывшейся закладке "Стороны" указать всех контрагентов-нерезидентов, являющихся сторонами по контракту.

Закладка «Доп.соглашения»

Закладка по умолчанию недоступна. С целью отслеживание взаиморасчетов по отдельным дополнительным соглашениям (спецификациям, инвойсам, приложениям и т.п.) к рамочному контракту ВЭД на закладке "Условия" следует установить флаг "Рамочный контракт (заключены доп. соглашения, спецификации и т.п.)" и в открывшейся закладке "Доп. соглашения" указать все элементы справочника "Договоры контрагентов", юридически являющиеся дополнительными соглашениями (спецификациями, инвойсами, приложениями и т.п.). Для облегчения работы с рамочными договорами рекомендуется в справочнике "Договоры контрагентов" для рамочных договоров создавать папки, имеющие те же имена, что и сами рамочные договоры, и размещать в этих папках как сам рамочный договор, так и все дополнительные соглашения (спецификации, инвойсы, приложения и т.п.) к нему. Программа автоматически обеспечит заполнение табличной части на закладке "Доп. соглашения" элементами справочника "Договоры контрагентов", находящимися в папке, одноимённой рамочному договору.

Закладка «Платежи»

На данной закладке указываются все платежные (банковские) документы, проходящие по контракту ВЭД. Итоговые суммы используются для контроля на превышение лимита для постановки на учет импортного контракта (3 млн. руб.), экспортного контракта (6 млн. руб.) и кредитного договора (3 млн. руб.), а также на превышение суммы контракта.

Закладка «Поставки (Отгрузки)»

На данной закладке при импорте указываются все документы поступления товаров и услуг, проходящие по контракту ВЭД, при этом закладка называется "Поставки". При экспорте на закладке указываются все документы реализации товаров и услуг, проходящие по контракту ВЭД, при этом закладка называется "Отгрузки". Итоговые суммы используются для контроля на превышение лимита для постановки на учет импортного контракта (3 млн. руб.) и экспортного контракта (6 млн. руб.), а также на превышение суммы контракта.

Закладка «Досье»

На данной закладке указываются все документы валютного контроля по контракту ВЭД. Причём к каждому документу валютного контроля можно прикрепить один или несколько файлов.

Закладка «Уступка»

Закладка предназначена для вывода в пункт 8 "8. Сведения о резиденте, которому уступаются требования (на которого переводится долг) по контракту" документа Заявления о постановке на учет контракта и отчета Ведомость банковского контроля данных о переуступке (переводе) долга по контракту, которые заполняются через документ Заявление в банк учета контракта с видом операции "Снятие с учета контракта".

Закладка «Кредит»

Закладка по умолчанию недоступна. Закладка используется для ввода данных по кредитному договору или договору займа, позволяя сформировать Заявление о постановке на учет кредитного договора. Для ее появления в шапке контракта следует установить флаг "Кредитный". На закладке "Кредит" по умолчанию имеются три вложенные закладки: "Условия", "Финансирование", "Погашение".

Вложенная закладка "Условия"

На закладке указываются основные условия получения кредита или получения (выдачи) займа для заполнения Заявление о постановке на учет кредитного договора.

Вложенная закладка "Финансирование"

На закладке указывается график получения основного долга по кредиту или получения (выдачи) основного долга по займу.

Вложенная закладка "Погашение"

На закладке указывается график возврата основного долга и уплаты процентов по полученному кредиту (займу) или возврата основного долга и получению процентов по выданному займу.

Вложенная закладка "Синдикат"

Закладка по умолчанию недоступна. Для ее появления на вложенной закладке "Условия" закладки "Кредит" следует установить флаг "Синдицированный (консорциональный) кредит/заём". На закладке "Синдикат" указывается перечень контрагентов-нерезидентов, предоставивших организации кредит (заём) на синдицированной (консорциональной) основе с указанием их долей в сумме кредита (займа).
