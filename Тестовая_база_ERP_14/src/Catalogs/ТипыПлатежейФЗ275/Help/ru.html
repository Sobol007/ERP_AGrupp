Типы платежей 275-ФЗ

Справочник содержит перечень предопределенных законодательством типов платежей, выполняемых с отдельных счетов, открытых в рамках выполнения государственного контракта по Государственному оборонному заказу.

Для типа платежей определяется соответствие видов подтверждающих документов, которые требуется оформить по данному типу платежа. Кнопка Добавить позволяет добавить прочие виды платежей в табличную часть соответствий.
