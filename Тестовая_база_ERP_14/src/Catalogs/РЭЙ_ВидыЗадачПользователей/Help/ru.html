Справочник «Источники задач ВЭД»

Справочник «Источники задач ВЭД» вызывается в интерфейсе "Внешнеэкономическая деятельность" из раздела меню «Настройки».

Наименования задач в справочник не подлежит редактированию пользователем и доступны только для просмотра, т.к. содержит только предопределнные значения элементов, служащих инициаторами порождения задач по ВЭД. Но пользователь может включать и отключать задачи, изменяя состояние флага Исп. (Использовать).
