Форма показателя

Показатели используются для распределения постатейных расходов между подразделениями в правилах и документах распределения постатейных расходов. Выбор показателя возможен при выборе способа распределения По разным базам для партий и подразделений.

Значения показателей задаются в форме Ввод показателей для распределения расходов на себестоимость продукции. Значения переменных показателей требуется вводить ежемесячно, постоянные показатели достаточно задать один раз, их значения будут использоваться до следующего изменения.
