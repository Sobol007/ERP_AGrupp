Справочник "Решения"

Назначение

Справочник предназначен для хранения решений базы знаний.
Решение базы знаний представляет собой вопрос и варианты ответа на него. Также для решении можно указать категории базы знаний, к которым относится данное решение, и другие решения, информация из которых может быть полезна.

Создается в подсистемах:

«Органайзер»
