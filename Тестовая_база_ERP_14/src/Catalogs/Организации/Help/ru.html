Организации

Справочник предназначен для ведения списка организаций, входящих в состав предприятия, и хранения постоянных сведений о них.
Организация может быть юридическим лицом, обособленным подразделением или индивидуальным предпринимателем. В справочнике непосредственно указываются обособленные филиалы, выделенные на отдельный баланс. Обособленные подразделения без выделения на самостоятельный баланс указываются в составе организации.

Параметры организации для целей внешнеэкономической деятельности

Для ввода дополнительных реквизитов, необходимых для осуществления внешнеэкономической деятельности, в карточке организации используется окно "Параметры ВЭД", вызываемое кнопкой "ВЭД". При первом открытии окна с параметрами ВЭД в него будут оттранслированы основные данные из карточки организации.

Некоторые банки требуют в Заявлении о постановке на учет контракта выводить адрес организации-резидента в том же регистре, в каком он содержится в Едином Государственном Реестре Юридических Лиц (Едином Государственном Реестре Индивидуальных предпринимателей). Для этого в окне "Параметры ВЭД" предумотрен флаг "Печатать адрес прописными буквами".

После любых изменений параметров ВЭД организации необходимо пересохранить и саму карточку организации, чтобы произведенные изменения вступили в силу.
