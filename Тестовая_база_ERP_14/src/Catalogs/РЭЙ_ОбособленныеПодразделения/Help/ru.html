Справочник "Обособленные подразделения ВЭД"

Справочник "Обособленные подразделения ВЭД" вызывается для ввода и редактирования информации в интерфейсе "Внешнеэкономическая деятельность" из раздела "Справочники".

Справочник предназначен для хранения списка обособленных подразделений организации для целей внешнеэкономической деятельности.

При формировании документа Заявление о постановке на учет контракта по обособленному подразделению следует установить флаг "КПП обособл. подразделения", указать данное подразделение, и в печатной форме Заявления о постановке на учет контракта будет отражен КПП обособленного подразделения.

Для организаций, являющихся крупнейшимии налоплательщиками, в справочник следует добавить фиктивное обособленное подразделение ВЭД с КПП крупнейшего налоплательщика.
