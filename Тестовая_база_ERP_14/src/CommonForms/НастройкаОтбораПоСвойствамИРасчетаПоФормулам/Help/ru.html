Расчет по формулам

Расчет по формулам доступен для вычисления:

  • количества материалов
  • количества побочного и промежуточного выхода
  • количества выходных изделий при разборке
  • трудозатат
  • времени работы видов рабочих центров

Можно задать произвольную формулу для расчета количества. В формуле можно использовать свойства характеристик продукции, свойства материала и свойства характеристик материала. Список свойств, доступных для выбора, ограничен перечнем дополнительных реквизитов числового типа.

Если характеристики у вида номенклатуры настроены как Общие с другим видом номенклатуры, то свойства таких характеристик так же можно использовать в формулах.

При проверке действующей спецификации производится контроль правильности указания операндов формул.

Отбор по свойствам

С помощью настройки потребления возможно задавать гибкие условия применения для материалов, побочного и промежуточного выхода и трудозатрат. Например, можно указать, что материал будет использоваться, когда свойство характеристики продукции имеет определенное значение.
