Документ "Заявление в банк учета контракта"

Документ вызывается для ввода и редактирования информации в интерфейсе "Внешнеэкономическая деятельность" из пункта меню "Заявление в банк учета контракта" раздела "Документы валютного контроля".

В настоящей версии программы документ "Заявление в банк учета контракта" предназначен для формирования четырёх видов заявлений, предоставляемых организацией-резидентом в уполномоченный банк учета контракта согласно Инструкции Банка России от 16.08.2017 №181-И "О порядке представления резидентами и нерезидентами уполномоченным банкам подтверждающих документов и информации при осуществлении валютных операций, о единых формах учета и отчетности по валютным операциям, порядке и сроках их представления":

  • Заявление о внесении изменений в раздел I ВБК;
  • Заявление об изменении кодов видов операций;
  • Заявление на предоставление ВБК;
  • Заявление о снятии с учета контракта.

Каждое из указанных заявлений формируется при выборе соответствующей операции документа:

  • Внесение изменений в раздел I ВБК;
  • Изменение кодов видов операций;
  • Предоставление ведомости ВБК;
  • Снятие с учета контракта.

Документ "Заявление о снятии с учета контракта"

Для формирования заявления о снятии с учета контракта следует использовать операцию "Снятие с учета контракта".

В шапке заявления указываются: организация-резидент, уполномоченный банк учета контракта и в качестве основания снятие с учета контракта - подпункты 6.1.1-6.1.6 и 6.9 Инструкции Банка России от 16.08.2017 №181-И:

  • 6.1.1 - перевод контракта на обслуживание в другой банк;
  • 6.1.2 - исполнение сторонами всех обязательств по контракту, вкл. исполнение третьим лицом;
  • 6.1.3 - уступка требования по контракту другому резиденту либо перевод долга на другого резидента;
  • 6.1.4 - уступка требования по контракту нерезиденту либо перевод долга на нерезидента;
  • 6.1.5 - исполнение (прекращение) обязательств по контракту по иным основаниям, не указанным в п/п 6.1.2;
  • 6.1.6 - прекращение оснований постановки на учет контракта, в т.ч. вследствие изменений и ошибочного принятия на учет;
  • 6.9    - снятие с учета контракта банком по истечении 90 календ. дней с даты завершения исполнения обязательств.

При выборе в качестве основания снятие с учета контракта подпункта 6.1.1 необходимо в поле "В другой банк" указать банк, в который переводится на расчетное обслуживание контракт ВЭД. Табличная часть на вкладке "Контракты" заполняется всеми теми контрактами ВЭД, которые переводятся на обслуживание в банк, указанный в поле "В другой банк".

При выборе в качестве основания снятие с учета контракта подпунктов 6.1.2-6.1.6 табличная часть на закладке "Контракты" заполняется всеми теми контрактами ВЭД, которые подлежат снятие с учета по разным основаниям.

Пункт 6.9 используется не для подачи заявления в уполномоченный банк УК, а для фиксации даты снятия контракта ВЭД с учета в банке УК. Табличная часть на вкладке "Контракты" заполняется аналогично пунктам 6.1.1-6.1.6.

В табличной части закладки "Документы" в произвольной форме указывается перечень документов, являющихся основаниями для снятие с учета контракта.

На закладке "Отв.лица" следует указать всех ответственных лиц по контракту:

  • Руководитель;
  • Главный бухгалтер;
  • Уполномоченный по учету контракта.

У всех ответственных лиц в справочнике Физические лица должны быть заполнены фамилия, имя и отчество. В карточке уполномоченного по учету контракта должен быть заполнен служебный телефон.

По заполнении всех необходимых полей заявление следует провести и распечатать (выгрузить в систему клиент-банк).

Поле "Дата снятия контракта с учета" заполняется после получения из банка УК информации о снятия контракта с учета. При указании даты  снятия контракта с учета в заявлении, эта дата появится на вкладке "Переоформление" документа Заявление о постановке на учет контракта, а сам документ будет заблокирован от изменений.

Документ "Заявление на предоставление ВБК"

Для формирования заявления на предоставления ведомости банковского контроля следует использовать операцию "Предоставление ВБК".

Порядок заполнение полей аналогичен документу "Заявление о снятии с учета контракта".

По заполнении всех необходимых полей документ следует провести и распечатать (выгрузить в систему клиент-банк).

Документ "Заявление об изменении кодов видов операций"

Для формирования заявления об изменении кодов видов операций следует использовать операцию "Изменение кодов видов операций".

Порядок заполнение полей аналогичен документу "Заявление о снятии с учета контракта".

По заполнении всех необходимых полей документ следует провести и распечатать (выгрузить в систему клиент-банк).

Документ "Заявление о внесении изменений в раздел I ВБК"

Для формирования заявления о внесении изменений в раздел I ведомости банковского контроля следует использовать операцию "Внесение изменений в раздел I ВБК".

Порядок заполнение полей аналогичен документу "Заявление о снятии с учета контракта".

В шапке заявления указываются: организация-резидент, банк учета контракта и в качестве оснований внесения изменений - подпункты 6.9, 7.1 и 7.9 Инструкции Банка России от 16.08.2017 №181-И:

  • 6.9 - внесение изменений в контракт либо продолжение исполнения обязательств после даты снятия с учета контракта;
  • 7.1 - внесение изменений и (или) дополнений в контракт, изменение иных сведений, содержащихся в разделе I ВБК;
  • 7.9 - пролонгация контракта без подписания дополнений и изменений, внесение изменений в раздел I ВБК для указания иной даты завершения исполнения обязательств.

По заполнении всех необходимых полей документ следует провести и распечатать (выгрузить в систему клиент-банк).

Таблица "Номер и дата переоформления контракта" на закладке "Переоформление" документа "Заявление о постановке на учет контракта" будет заполнена данными из заявления в банк.
