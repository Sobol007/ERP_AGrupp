Сопоставление 1С:Номенклатуры

В форме производится поиск и сопоставление публикуемых товаров и карточек сервиса 1С:Номенклатура. После сопоставления и публикации, товар автоматически распределяется по категориями сервиса 1С:Номенклатура, при открытии карочки торгового предложения покупателем отображаются данные карточки 1С:Номенклатура (свойсво, изображение).

При открытии формы автоматически выполняется поиск эталонов. Если эталоны не найдены, пользователь может выбрать данные вручную. После проверки сопоставления необходимо установить флаг подтверждения для строк и выполнить команду Сохранить сопоставления, при этом подтвержденные строки удаляются. Оставшиеся строки можно сопоставить позже.

Возможные отборы при сопоставлении:

  • Вся номенклатура - отображение всей номенклатуры сопоставления.
  • Точное совпадение - автоматически сопоставленные карточки сервиса 1С:Номенклатуры по ключевым реквизитам (штрихкод, артикул).
  • Неточное соответстие - требуется выбора варианта сопоставления.
  • Совпадение не найдено - строки, для которых в сервисе 1С:Номенклатура не найдены карточки.
  • Скрывать сопоставленные - скрытие строк, для которых выполнено автоматическое или ручное сопоставление с эталонами.
