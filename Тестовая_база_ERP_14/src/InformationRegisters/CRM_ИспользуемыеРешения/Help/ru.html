Регистр сведений "Используемые решения"

Назначение

Регистр сведений предназначен для хранения взаимосвязей решений базы знаний с документами, в которых они используются.

Создается в подсистемах:

«База знаний»

Измерения:

Объект.

Документ, который использует решения (События, Исходящие электронные письма)

Решение.

Решение, с которым устанавливается взаимосвязь (Решения)

Ответ.

 Ответ, который выбрал пользователь из списка ответов данного решения (Число)

Ресурсы:

Пользователь.

Пользователь, который добавил использование решения (Пользователи)
