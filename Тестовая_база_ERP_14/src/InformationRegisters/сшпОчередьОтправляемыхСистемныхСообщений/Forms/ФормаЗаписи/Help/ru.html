Запись регистра сведений "Очередь отправляемых сообщений" предназначена для хранения информации о сообщении, отправляемом в ESB.

Для каждого сообщения указывается:

  • Идентификатор сообщения - уникальный идентификатор отправляемого сообщения (тип - Уникальный идентификатор);
  • Хранилище - сами данные, передаваемые в сообщении (тип - Хранилище значения);
  • Дата регистрации - дата, когда отправляемое сообщение было зарегистрировано в очереди (тип - Дата);
  • Идентификатор базового сообщения - уникальный идентификатор исходящего сообщения, прородителя отправляемого сообщения (тип - Уникальный идентификатор) ;
  • Статус - текущий статус отправляемого сообщения (тип - перечисление "Статусы соощений"). Может принимать значения:

Новое - сообщение только что поступило в систему;

Ожидание обработки - сообщение ожидает обработки;

В обработке - сообщение находится в обработке;

Отсутствует обработчик - для данного класса сообщения не найден обработчик;

Обработано - сообщение обработано;

Ошибка обработки - при выполнении обработчика возникла ошибка;

Обработка отменена - обработка сообщения отменена;

Ожидание отправки - сообщение ожидает отправки (для исходящих сообщений);

Отправлено - сообщение отправлено (для исходящих сообщений);

Отправка отменена - отправка сообщения отменена (для исходящих сообщений).
